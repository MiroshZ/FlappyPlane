# FlappyPlane

Небольшая кооперативная 2D-игра на двоих на Python для macOS: самолёт пролетает через колонны, собирает монеты, зарабатывает валюту и открывает новые скины.

## Что внутри

- `pygame`
- запуск одной командой через существующий `.venv`
- полноэкранный режим
- главное меню
- магазин скинов за монеты
- пауза по `Esc`
- сохранение прогресса между запусками
- очки, лучший результат и collectible-монеты

## Управление

- `Enter` или `Space` в главном меню - начать забег
- `W` прыжок вверх для первого игрока
- `Arrow Up` прыжок вверх для второго игрока
- левая кнопка мыши
- `Esc` во время игры - пауза / продолжить

После проигрыша:

- `R`
- `Space`
- клик мышью

В магазине:

- клик по карточке скина - купить или надеть
- `Esc` или `M` - назад в меню

## Структура

- `main.py` - точка входа
- `game/app.py` - состояния игры, меню, магазин, игровой цикл
- `game/entities.py` - самолёт, препятствия, облака, монеты
- `game/settings.py` - настройки окна и баланса
- `game/save_data.py` - сохранение монет, рекорда и открытых скинов
